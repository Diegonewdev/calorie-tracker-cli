Calorie Tracker CLI

Log meals & calculate daily calories/macros — fast and offline.

Why?

Simple CLI for quick meal tracking, perfect for busy days on the road.

How to run

python src/main.py

Features

  • Add meals with grams
  • Auto-calc kcal, protein, carbs, fat
  • CSV output for portability
